# Каталог ИП «ТехноХолод»

Файлы каталога кондиционеров для сайта tehnoholod369.kz.

- `katalog-bytovye.html` — бытовые кондиционеры (306 моделей, 15 брендов).
- `katalog-poluprom.html` — полупромышленные: кассетные, канальные, напольно-потолочные, колонные (340 моделей, 10 брендов).
- `img/realcon/` — фотографии моделей AERONIK / CHERBROOKE / GREEN.

На сайте /katalog стоит один небольшой блок-хаб (выбор направления), который
по клику подтягивает нужный каталог с CDN jsDelivr.
